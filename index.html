Datos Generales del Curso

Curso: Desarrollo web

Descripción: Este curso es el primero en la secuencia de desarrollo web y cubre los fundamentos básicos de los lenguajes HTML, CSS y JavaScript, que son los elementos esenciales de cualquier aplicación web.
A través del dominio de los elementos del lenguaje HTML y el uso de los elementos de estilo en lenguaje CSS los estudiantes podrán diseñar sitios web atractivos y funcionales. Mediante el uso de los conceptos básicos y sintaxis de JavaScript, los estudiantes podrán crear aplicaciones interactivas de front-end e introducir contenido dinámico.
